<h1>TLDR Extension</h1>

This extension allows you to get a summary of a site with just a few clicks.
It uses AI and you can ask for it for more details about a certain part of a site and it can respond with text and also showing you a quote from the page about your question.
Uses openrouter api for AI. You can get a key for it at openrouter.ai for free.


https://github.com/user-attachments/assets/789094d1-6dae-4c9c-80db-d90250e733cd

<h2>To install:</h2>

-[Download](https://github.com/Yusufumben08/TLDR/archive/refs/heads/main.zip) the repository

-Extract it to a folder

-Type "chrome://extensions/" in the search bar and press enter

-Enable developer mode in top right

-Press "Load unpacked extension"

-Select the folder you extracted in step 2

Done! Now pin the extension to the search bar and click on it to use it whenever you'd like!
It will ask for an api key for the first time, you can get it from openrouter.ai and enter it. once you enter it and press the button close the extension and reopen it.
